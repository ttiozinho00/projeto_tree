# projeto_tree

- metodo de compilar

```
gcc -o tree tree.c funcoes.c -Wall -pedantic -Wextra -Werror
```

-  Execução Padrão (Somente Diretórios)

```
./TREE++
```

- Exibir Ajuda (/?)

```
./TREE++ /?
```

- Modo ASCII (/A)

```
./TREE++ /A
```

- Listar Arquivos (/F)

```
./TREE++ /F
```

- Combinação de ASCII e Listagem de Arquivos

```
./TREE++ /A /F
```
